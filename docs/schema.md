# Schema   
from: [react-jsonschema-form](https://rjsf-team.github.io/react-jsonschema-form/)   
Three types of schema
## JSON Schema  
Indicates what data the form needs to collect.    
## UI Schema
Indicates what type of data fields are to be used to collect the data.   
## formData    
Provides some default (or other) data with which the form should be populated on load.
